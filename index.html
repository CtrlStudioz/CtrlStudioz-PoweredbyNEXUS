IoT • Forest Protection • Real-time Monitoring

GreenWard (TerraVerde): IoT Forest Protection System

An IoT-powered system designed to prevent illegal logging and detect forest fires in real-time. Uses sound, smoke, and fire sensors with LoRa transmission to remote monitoring stations.

Key Features:

  • Sound detection for illegal logging activities with real-time alerts
  • Early fire detection with smoke and temperature sensors
  • GPS location tracking for precise incident mapping
  • SMS alerts via SIM800L GSM module to authorities
  • LoRa long-range communication for remote forest areas
  • Real-time web dashboard with live location tracking
